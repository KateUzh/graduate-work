# Graduate Work

## Описание проекта
Веб-приложение для размещения объявлений с возможностью регистрации пользователей,
авторизации, управления объявлениями и комментариями.

В приложении реализована ролевая модель доступа:
- `USER`
- `ADMIN`

## Используемые технологии
- Java 11
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security
- PostgreSQL
- Liquibase
- OpenAPI (Swagger)
- MapStruct
- Docker

## Функциональность
- регистрация и авторизация пользователей
- получение и обновление профиля пользователя
- смена пароля
- просмотр объявлений без авторизации
- создание, редактирование и удаление объявлений
- работа с комментариями
- разграничение прав доступа (USER / ADMIN)

## Запуск проекта
1. Установить PostgreSQL
2. Создать базу данных
3. Настроить файл `application.yml`
4. Запустить приложение

## Документация API
Swagger-документация доступна по адресу:  
`http://localhost:8080/swagger-ui.html`
